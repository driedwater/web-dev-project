Quadratic Equation

Quadratic equations are expressed in this form:

\[ax^2 + bx + c\]

Where a, b and c are constants.

Example:

\(2x^2 + x - 1 = 0\)

As shown above, a quadratic equation is when a equation contains \(x\) to a degree of 2.

Degree of 2 mean the power of \(x\) is a 2.

When we have \(x + 1 = 2\) where \(x\) is only a power of 1, it is called a linear equation.

How do we get a quadratic expression?

We multiply 2 linear expressions together.

\(x(x + 1) = 2x^2 + x\)

or \((2x - 1)(2x + 1) = 4x^2 - 1 \) etc

Solving quadratic equations

\[2x^2 + x = 1\]

We cannot factor out the x and divide it.

\(x(2x + 1) = 1\)

\(2x + 1 = \frac{1}{x}\)

Instead, we should factorise it by grouping. This is because some quadratic equations can be factorised out into 2 linear expressions.

\(2x^2 + x - 1 = 0\)

As show above, first, we should reduce \(x\) into 2 parts.

Then, for the constant, we make sure it is \(-1\) by multiplying -1 and 1

Lastly, we do cross multiplication from \(x\) to \(-1\) and \(2x\) to \(1\)

we get:

\((2x-1)(x+1) = 0\)

We solve each linear equation individually:

\(2x - 1 = 0 \)

\(x = \frac{1}{2}\)

\(x + 1 = 0\)

\(x = -1\)

Therefore, there is 2 solutions to this quadratic equation. \(x = \frac{1}{2}\) and \(x = -1\)

Not all quadratic equations can be solved using real numbers.

Some quadratic equations such as:

\(x^2 + 1 = 0\)

cannot be solved using real numbers as they do not have real number solutions. Instead, it requires complex numbers in order to solve it. This is because if we try to solve it:

\(x^2 = -1\)

we get this equation:

\(x = \pm\sqrt{-1}\)

There is no real number representation of \(\sqrt{-1}\). \(\sqrt{-1}\) is defined as \(i\). However, we will not be learning complex numbers for now.

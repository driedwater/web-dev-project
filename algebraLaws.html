Introduction to Variables

What is a Variable?

\begin{equation} x = 2 \end{equation}

A variable is similar to a storage that stores information. We can define other alphabets such as a, b, c and etc.

Adding and substraction of variables

\begin{equation} 2x + x \end{equation}

If we were to add these two variable together, it will equate to \(3x\)

How does it work?

First, we will factorise out x.

\(x(2 + 1)\) [Note that x = 1x, we do not write out the 1 as it is implied.]

Then, we add the numbers together.

\(x(3)\)

[You can always skip these steps, this is just an example / way to think of adding and subtracting algebra]

Lastly, we remove the brackets and write out the numbers first, followed by the alphabet.

\(3x\)

Similarity, subtraction works the same way, but instead of adding, we subtract the numbers.

\(3x - x = 2x\)

Question: Can we add \(3x^2 + x\) ?

Answer

No. If we were to factorise out x, we get \( x(3x + 1) \)

As \(3x\) and \(1\) are unlike terms, we are unable to add them together.

Multiplication and Division of variables

\(2 \times x = 2x\)

\(2x \times 3x = 6x\)

Similarly, \(3x \times 2 = 6x\) as well.

\(2x \times 2x = 4x^2\)

we separate each components and multiply them.

Division

\(3x \div 3 = x\)

Similar to multiplication, we separate each component and divide them.

\(3x \div 3x = 1\)

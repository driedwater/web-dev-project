Linear Equation

Linear equations are expressed in this form:

\[ax + b\]

Solving Linear equations

\[2x + 1 = 3\]

We want to solve for \(x\)

First, we should subtract 1 from the equation.

\(2x + 1 - 1 = 3 - 1\)

We are left with:

\(2x = 2\)

now divide the equation by 2.

\(2x \div 2 = 2 \div 2\)

we can reduce this to:

\(x = 1\)

Therefore, \(x = 1\)

Questions

\(2(2x + 6) = 12\)

\(2(\frac{1}{2}x + 4) = 5\)

\(\frac{1}{3}(2x + 3) = 15\)

Answers

\(x = 0\)

\(x = -3\)

\(x = 21\)
